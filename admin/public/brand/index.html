Vector · transparent
The big PNGs are not sharper, only smoother. The original artwork is 439×410 pixels, so anything above that is upscaled. For anything printed large, use the SVG.
The SVG has flat gold instead of a gradient. No vector original exists, so this was traced from the image. It scales cleanly, but the gradient on the price tag became solid gold. If the designer still has the Illustrator or Figma file, export from there instead.
The white outline is part of the logo, not background. The mark has a white keyline around it and a hole through the bag handle. Cutting out every white pixel would have eaten the outline too, so only the background beyond the keyline is transparent.
